Serveur personnel

Pour mes besoins personnels et mon plaisir je loue un serveur dédié chez OVH afin d'auto-herberger les services dont j'ai besoin. Je vous tout d'abord avoir une solution de cloud personnel qui aie les mêmes fonctionnalités que les cloud connus (Google Drive, OneDrive, ICloud...) mais où je reste maître de mes données.

J'avais pour cela besoin d'une solution qui me permettait de gérer :

  • mes fichiers
  • mes contacts
  • mes calendriers
  • mes listes de taches
  • mes mots de passe

Client: Moi !
Nature du projet :

Hébergement d'un cloud personnel

Stack technique :
Docker
MariaDB
PHP
VPS OVH

Nextcloud : la solution complète

La solution la plus complète que j'ai pu trouver est Nextcloud. Il embarque toute ses fonctionnalités, est extensible et des clients sont développé pour PC et Android.

Pour installer Nextcloud, j'ai utilisé l'image Docker dans un docker-compose. Ce docker-compose contient un container Nextcloud ainsi qu'une base de données MariaDB.

Ajout d'autres services

Par la suite j'ai voulu héberger d'autres services comme Photoprism pour gérer ma phototèque ou encore Jellyfin pour gérer mes medias vidéos.
